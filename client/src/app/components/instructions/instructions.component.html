Genome Context Viewer

A web-app that visualizes genomic context data provided by third party services in a single, federated interface by using gene-family assignment as a unit of search and comparison.

Search View

Search across multiple data providers for genomic contexts similar to your query and align the results to the query in a manner that emphasizes structural variation.

Put the results into context with chromosome-scale synteny blocks computed on-demand across data providers.

And better understand local and global strucuture with pairwise gene-loci dot plots.

Multi View

Retrieve multiple genomic contexts from different data providers, group them based on shared gene-family content, and perform a multiple sequence alignment of each group that emphasizes structural variation.

Put the results into context with chromosome-scale synteny blocks computed on-demand across data providers and displayed in a Circos-style viewer.
